## 🚀 GitHub Pages पर Deploy करें

1. इस repository को **fork** या **clone** करें
2. `Settings` → `Pages` में जाएँ
3. **Source:** `main` branch, `/ (root)` चुनें
4. `Save` करें → आपकी साइट live हो जाएगी!
